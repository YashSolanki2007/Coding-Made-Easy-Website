This Webpage will show you the basics of image classification in neural networking.



The most common types of neural networks are called fully connected neural networks. These are made up of three different layers. The first layer or the input layer, the second layer or the hidden layer and the third layer or the output layer. The image below shows a fully connected neural network.


However this type of a neural network is not good for finding out patterns in images for such a case another type of a neural network called a Convulutionary Neural Network or CNN is used.


However you can only use a CNN if the image is a four dimensional image. Because otherwise you will get an error. You can easily use the python module numpy to reshape your images. But images do not have a fourth dimension so in place of the fourth dimension you have to add one.
